**Firebase Contact Pro**
A high-performance, real-time Contact Management application built with React, Tailwind CSS, and Firebase. This app allows users to create, search, update, and delete contacts with instant database synchronization and a smooth UI.

** Features**
Real-time Database: Powered by Firebase Firestore for instant updates across all clients.

Full CRUD Functionality: Seamlessly Create, Read, Update, and Delete contacts.

Instant Search: Filter through your contact list in real-time as you type.

Form Validation: Robust client-side validation using Formik and Yup.

Elegant UI: Responsive design crafted with Tailwind CSS.

Interactive Notifications: Beautiful toast alerts provided by React Hot Toast.
